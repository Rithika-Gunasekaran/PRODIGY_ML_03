Cats vs Dogs Classifier (SVM)

This is a beginner-friendly Machine Learning project that classifies images of cats and dogs using a Support Vector Machine (SVM).

---

🚀 Project Overview

This project demonstrates how images can be converted into numerical data and used to train a machine learning model for classification.

---

🧠 How it Works

- Load images from dataset
- Resize images to 64x64
- Convert images into numerical arrays
- Train SVM model
- Predict and calculate accuracy

---

📂 Dataset

- Custom dataset with small number of images
- Categories:
  - Cats 🐱
  - Dogs 🐶

---

🛠️ Technologies Used

- Python
- OpenCV
- NumPy
- Scikit-learn

---

▶️ How to Run

1. Install required libraries:

pip install numpy opencv-python scikit-learn

2. Run the program:

python main.py

---

📊 Output

- Displays accuracy of the model
- Example:

Accuracy: 0.7
